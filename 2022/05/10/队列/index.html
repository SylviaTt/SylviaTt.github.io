队列

2022-05-10 数据结构 评论

队列的结构体：1. 队头指针；队尾指针；2.数据元素

队列初始化:初始化队头指针和队尾指针为0

队列为空:队尾指针 = 队头指针

队列已满：(Q.rear + 1) % MAXSIZE == Q.front

入队：1.判满；2.加入队尾;3.更新队尾指针：Q.rear = (Q.rear + 1) % MAXSIZE

出队：1.判空；2.用x返回队头元素；3.队头指针后移Q.front = (Q.front + 1) % MAXSIZE；

打印队列：1.判空；2.遍历（更新front）

获取队首元素：输出 Q.data[Q.front]

清空队列：队头队首指针置0
